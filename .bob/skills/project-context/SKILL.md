# Project Context Analysis

## Purpose
Analyze any codebase to understand its conventions, architecture,
and tribal knowledge so that AI coding agents can be more effective.

## Steps
1. Read the file tree to understand project structure
2. Identify the tech stack from config files
3. Analyze naming patterns from code samples
4. Extract architecture from directory organization
5. Find build/test commands from package.json or equivalent
6. Summarize findings as structured context
